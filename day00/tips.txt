touch -t 06012342 file_name = change le timestamp du fichier. 

Si fichier dans le futur, alors l'année est affichée. Il faudra modifier la date du système :)

Pour l'affichage, changer le format dans les parametres systèmes.

Droit : 
CHMOD xxx file_name.
4 : Read
2 : Write
1 : Execute

ln -s link(destination) fichier_a_creer = lien symbolic

touch -h -t 06012220 test6 pour forcer le changement d'heure d'un fichier pointant vers un autre.
